## Manual Research Suggestions

In addition to the 13 automated opportunities, the system now generates 7 manual research methods per keyword.

### Method 1: Guest Post Search
**Description:** Find blogs accepting guest posts about your keyword
```
Search queries:
- "[keyword] + write for us"
- "[keyword] + guest post"
- "[keyword] + contribute"
- "[keyword] + submit article"
- "[niche] + guest author"

Expected results: 15-30 opportunities
Effort level: Medium
Success rate: 35%
```

**Example for "task management software":**
- Search: `"task management software" + "write for us"`
- Results: TaskProject.com accepts guest posts, ProdFlow.org has contributor program, etc.

### Method 2: Competitor Backlink Analysis
**Description:** Find who links to top competitors, contact them
```
Steps:
1. Find top 3-5 competitors ranking for keyword
2. Use free Ahrefs/Semrush tier to see their backlinks
3. Note sites linking to competitors
4. Reach out for backlinks

Expected results: 10-25 opportunities
Effort level: High
Success rate: 45% (proven tactic)
```

**Why it works:** If Site A links to your competitor, they're interested in your niche. More likely to link to you too.

### Method 3: Q&A Site Mining
**Description:** Answer questions on Quora, Reddit, Stack Overflow with links to your site
```
Platforms:
- Quora: 300M+ monthly users
- Reddit: 430M+ monthly users
- Stack Overflow: 14M+ developers
- Answer.com: Similar to Quora

Expected results: 20-50 opportunities
Effort level: Low (just answer questions)
Success rate: 25% (depends on content quality)
```

**Why valuable:** Even though link placement is indirect (user profile, comment), Q&A sites have massive traffic and authority.

### Method 4: Resource/Roundup Posts
**Description:** Find resource pages and "best of" lists that should include you
```
Search queries:
- "[keyword] + resource"
- "[keyword] + tools"
- "[keyword] + list"
- "best [keyword]"
- "top 10 [keyword]"

Expected results: 25-40 opportunities
Effort level: Medium
Success rate: 40%
```

**Example:** User created task management tool â†’ Search "best task management tools" â†’ Find 30 roundup posts that list similar tools â†’ Pitch yours.

### Method 5: Industry Directories
**Description:** Submit to directories and resource libraries for your industry
```
Search for:
- "[niche] + directory"
- "[niche] + resource library"
- "[niche] + tools directory"
- "[niche] + submission"

Expected results: 5-15 opportunities
Effort level: Low (simple submission forms)
Success rate: 50% (highest success rate!)
```

**Why high success rate:** Directory owners WANT submissions. Easy acceptance.

### Method 6: Broken Link Building
**Description:** Find broken links and suggest your content as replacement
```
Steps:
1. Install "Check My Links" Chrome extension
2. Browse high-authority sites in your niche
3. Identify broken links (404 errors)
4. Create content that replaces the broken link
5. Email site owner with suggestion

Expected results: 10-20 opportunities
Effort level: Very High (requires content creation)
Success rate: 60% (highest success rate if done right!)
```

**Why high success rate:** You're solving their problem (fixing broken link). They're motivated to respond.

### Method 7: HARO (Help A Reporter Out)
**Description:** Get quoted in articles, receive backlinks from journalists
```
Steps:
1. Sign up for HARO (helpareporter.com)
2. Watch for requests matching your keyword
3. Provide expert answers to journalists
4. Get featured in published articles with backlinks

Expected results: 2-5 opportunities
Effort level: Medium (writing good answers)
Success rate: 30%
```

**Why valuable:** Journalistic links = high authority, professional credibility

### New Function: calculateReachScore()
```javascript
const calculateReachScore = (siteDA, minDA, maxDA) => {
  // Scores sites by how reachable they are for user's DA
  if (siteDA <= maxDA) return 1;        // Achievable
  if (siteDA <= maxDA * 3) return 2;    // Reach
  return 3;                              // Aspirational
};
```
